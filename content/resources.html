--- title: "Resources" ---

Here are some great resources to learn about the topics covered in Data Science and Programming.

STAT 545

Jenny Bryan

The course upon which this course was originally based. Covers some topics we won’t get to.

R for Data Science (aka “r4ds”)

Garrett Grolemund, Hadley Wickham

Closely mirrors the topics of this course, making this book a good alternative to the main guidebook for the course.

Happy git with R and RStudio

Jenny Bryan, the STAT 545 TAs, Jim Hester

Helps you set up some key tools we’ll be using for the course.

What They Forgot to Teach You About R

Jennifer Bryan, Jim Hester

A work-in-progress, this book helps you use R responsibly for data analysis.

Advanced R

Hadley Wickham

If you want to learn more about R as a programming language, this is a very readable and concise way of doing so.

ggplot2 book

Hadley Wickham

Useful for digging deeper into ggplot2.

RMarkdown book

Yihui Xie, J. J. Allaire, Garrett Grolemund

Comprehensive resource for everything R Markdown related.

R packages

Hadley Wickham

Readable, approachable help for making R packages.
